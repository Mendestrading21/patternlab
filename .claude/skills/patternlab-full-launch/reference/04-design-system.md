# 04 — Design system

## Direction

- sombre premium ;
- surfaces graphite/bleu nuit ;
- vert énergique ;
- rouge chaud ;
- or pour les récompenses ;
- ciel et paysages pour les parcours ;
- typographie très lisible ;
- cartes arrondies ;
- ombres douces ;
- profondeur contrôlée.

## Tokens sémantiques

- `success`
- `danger`
- `warning`
- `info`
- `bullish`
- `bearish`
- `neutral`
- `reward`
- `surface`
- `surfaceElevated`
- `textPrimary`
- `textSecondary`
- `border`

Ne pas utiliser vert = toujours correct et rouge = toujours faux dans les exercices de marché : distinguer sémantique pédagogique et sémantique financière.

## Accessibilité

- contraste AA ;
- zones tactiles ;
- tailles dynamiques ;
- lecteurs d’écran ;
- retour haptique non exclusif ;
- réduire les animations ;
- ne jamais transmettre une information uniquement par couleur.
