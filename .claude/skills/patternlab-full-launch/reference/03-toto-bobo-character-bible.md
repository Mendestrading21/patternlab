# 03 — Bible de Toto et Bobo

## Toto

- espèce : taureau ;
- couleur : vert ;
- rôle : curiosité, action, hypothèse haussière ;
- caractère : joyeux, spontané, encourageant ;
- faiblesse : excès de confiance ;
- accessoires possibles : lunettes, livre, loupe, tablette, trophée.

## Bobo

- espèce : ours ;
- couleur : rouge ;
- rôle : prudence, risque, hypothèse baissière ;
- caractère : sceptique, concentré, protecteur ;
- faiblesse : excès de prudence ;
- accessoires possibles : livre de risque, carnet, écrans, bandeau d’entraînement.

## Relation

Ils ne sont pas ennemis. Ils représentent deux lectures complémentaires du marché.

Exemple :

- Toto : « Le mouvement semble puissant. »
- Bobo : « Oui, mais vérifions le volume et l’invalidation. »

- aucune violence ;

- aucun langage de pari ;

- pas de personnage qui remplace l’explication ;
- ne pas mettre les mascottes sur chaque écran ;
- ne pas mélanger les noms Toro/Bruno avec Toto/Bobo.
