# Installation — PatternLab V5 Master Skill

Copier le dossier complet dans :

```text
<racine-patternlab>/.claude/skills/patternlab-v5-master/
```

Le fichier principal doit être ici :

```text
.claude/skills/patternlab-v5-master/SKILL.md
```

Dans Claude Code, à la racine de PatternLab, coller le contenu de :

```text
prompts/START-PLAN-THEN-EXECUTE.md
```

Claude doit d’abord créer le plan complet dans `docs/`, puis commencer immédiatement l’exécution du Lot 0 et continuer lot par lot.
