- comparaison haussier / baissier / neutre ;

## À ne pas copier

- mise en page exacte ;
- textes et slogans ;
- BUY / SELL ;
- TP / SL comme consignes ;
- promesses de profit ;
- formulations de certitude ;
- esthétique néon excessive.
