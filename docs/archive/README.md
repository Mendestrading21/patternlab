# Archives du projet

Ce dossier conserve la mémoire utile sans polluer les instructions et sources actives.

- `agent-programs/` : anciens skills et prompts Claude terminés ; lecture historique uniquement.
- `PROJECT_STATUS_ARCHIVE.md` reste à la racine de `docs/` pour préserver ses liens ADR existants.

Une archive n’est jamais une source de vérité. Pour travailler, revenir à `CLAUDE.md`,
`docs/CURRENT_STATE.md`, `docs/REPOSITORY_MAP.md` et `docs/DECISIONS_INDEX.md`.
