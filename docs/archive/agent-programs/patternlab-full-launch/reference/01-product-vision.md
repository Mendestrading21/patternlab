# 01 — Vision produit

## Promesse

> Apprends à lire, comprendre et analyser les marchés en quelques minutes par jour.

## Positionnement

PatternLab est un jeu d’apprentissage sérieux. Il doit donner envie de revenir, mais sa valeur vient de la maîtrise réelle des concepts.

## Public initial

- débutants ;
- étudiants ;
- jeunes actifs ;
- investisseurs particuliers ;
- utilisateurs intermédiaires qui révisent leurs bases.

## Piliers

1. Apprendre
2. Analyser
3. S’entraîner
4. Progresser

## Boucle quotidienne

1. Mission du jour
2. Micro-leçon
3. Exercices
4. Feedback
5. XP
6. Révision d’une faiblesse
7. Progression visible
8. Invitation à revenir

## Différenciation

- graphiques réellement interactifs ;
- patterns financiers ;
- progression par compétences ;
- erreurs expliquées ;
- contenu WMB vérifié ;
- Toto et Bobo ;
- univers visuel premium ;
- exercices plus riches que des QCM.

## Ce que le produit n’est pas

- courtier ;
- plateforme de signaux ;
- portefeuille réel ;
- terminal professionnel ;
- réseau social spéculatif ;
- casino ;
- copie d’une autre application.
