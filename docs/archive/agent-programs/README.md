# Programmes agent archivés

Ces programmes ont servi à construire les versions précédentes de PatternLab. Ils sont conservés
pour expliquer l’historique des décisions, mais sont **inactifs**.

Claude Code ne doit ni les charger automatiquement, ni exécuter leurs commandes de publication, ni
les recopier sous `.claude/skills/`. Le seul skill actif est
`.claude/skills/patternlab-learning-master/`.

Les images de référence brutes aux licences incertaines ne sont pas conservées dans l’archive ; seuls
leurs principes originaux et décisions restent documentés.
